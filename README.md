# MCBE-Spammer

![image](https://i.ibb.co/qrqDShG/image.png)

## Installation

- Installer nodeJS : https://nodejs.org/
- Lancer `install.cmd`

## Utilisation

- Modifier la liste des pseudos de vos comptes, l'ip, et le port dans `main.js`
- Lancer `start.cmd`
